# Beacon

Campus Intelligence OS for Cursor's Campus Lead role.

Three layers: Ambassador Operations, Community Operations, Strategic Intelligence. See `SPEC.md` for the full feature breakdown, `SCHEMA.md` for the data model, `PIPELINE.md` for the flow.

## Development

```bash
pnpm install
pnpm dev
```

See `.cursor/rules/stack.md` for environment variables and commands.

## Status

In development. Day 1 demo targets 5 schools (MIT, Stanford, CMU, Berkeley, Columbia) with 20-25 real professors.
